macbook免密ssh

发表于 2023-02-24 更新于 2023-09-21 分类于 MacBook Disqus：

win 下使用xshell等软件连接服务器很便捷，换了Mac之后，感觉terminal就可以完成目前我的全部需求，这种情况下，使用公私钥对登陆，很方便、很有必要！

配置公私钥对


cd ~/.ssh

ssh-keygen -t ed25519 -C "Comment"

ssh-copy-id -i 公钥文件名 root@服务器ip

ssh-add -K 私钥文件名
  1. 进入ssh配置文件夹下
  2. 生成公私钥对
  3. 把公钥上传到服务器
  4. 把私钥添加到ssh列表中

注：「有些mac第三步之后就可以登陆了」
